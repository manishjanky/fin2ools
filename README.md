# [fin2ools](https://fin2ools.vercel.app/)

This application provides you with some financial tools that can help you with your day to day financial needs in a secure manner. For e.g. tracking your investments in mutual funds, calculating returns on FD, RD, PPF, etc,


## Features
- Mutual Fund Portfolio Tracker
- FD, RD, PPF Returns Calculator
- Secure and Offline: All data is stored locally in the browser localStorage, ensuring your financial information remains private and secure.
- Non-intrusive: The application is designed to be lightweight and non-intrusive, It does not require any personal information or registration.
- PWA Support: The application is a Progressive Web App (PWA), which means you can install it on your device and use it offline, just like a native app.
